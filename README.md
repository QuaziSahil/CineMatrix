## Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Data APIs**: OMDB (IMDB) Primary, TMDB Secondary
- **Export Engine**: html2canvas for high-fidelity image rendering

## Usage

1. Open `index.html` (or access via local dev server)
2. Search for any TV series
3. Select a result to generate the ratings matrix
4. Toggle seasons or download the full show visualization

## Project History
Originally developed as "SeriesGraph", rebranded to **CineMatrix** to reflect the unique data-matrix visualization style.
